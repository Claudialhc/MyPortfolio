MovieTaste APP

This app was created using Bulma.io, JavaScript and two APIs. The two API's used are Giphy API and OMDB API. The purpose for this app is to help users find entertainment during their lockdown boredom. The user will be asked to enter the name of a the movie they are searching for, when they type the title of the movie the app will display their choice of movie on the left column and on the right column the Giphy associated with the movie title will appear. OMDB API is used to populate the user's movie choice with the movie's poster, release year, rating, actors and short plot. The Giphy API will then display Gifs associated with that movie title.
